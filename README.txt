
# Accesorios La Plata - Gestión de Pedidos (Preliminar)

## Cómo levantar el proyecto

1. Instalá Node.js si no lo tenés: https://nodejs.org
2. Abrí una terminal y navegá a la carpeta del proyecto.
3. Ejecutá:

    npm install

4. Después levantá el proyecto con:

    npm run dev

5. También podés subirlo a https://vercel.com creando una cuenta gratis y conectando este proyecto.

## Notas

- Este ZIP es una versión preliminar para pruebas.
- El flujo incluye: Inicio, Nombre de cliente, Grilla de modelos con buscador, Selección de accesorios, Más accesorios, Resumen.
- Los pedidos se guardan temporalmente en LocalStorage (sin backend todavía).
